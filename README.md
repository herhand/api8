# Laravel 8 Passport
Basic API based on Laravel 8 Passport

## Source
* https://www.tutsmake.com/laravel-8-rest-api-crud-with-passport-auth-tutorial/
* https://laravel-news.com/api-logger-package

## Installation Guide
    1)Clone Or Download The Repository

    2)Set the .env File with the relevant configuration

    2)Goto to project root and open a Terminal Enter Following Commands 
        * composer install
        * php artisan migrate
        * php artisan passport:install
